# 🚗 Car Price Prediction With Linear Regression

This project predicts the selling price of a car based on its features using a **Linear Regression model with preprocessing pipeline**.

---

## 📌 Project Overview
The project includes:
- Data cleaning and preprocessing
- Feature scaling and encoding
- Model training using Linear Regression
- Model evaluation
- Visualization of predictions
- User input for real-time prediction

---

## 🧠 Key Features
- 🔄 Pipeline (Preprocessing + Model)
- 📊 One-Hot Encoding for categorical data
- ⚖️ Feature scaling using StandardScaler
- 📈 Actual vs Predicted visualization
- 🔮 User input prediction system

---

## 📂 Dataset
- File: `car data.csv`
- Important Columns:
  - Year
  - Present_Price
  - Driven_kms
  - Fuel_Type
  - Selling_type
  - Transmission
  - Owner
  - Selling_Price (target)
